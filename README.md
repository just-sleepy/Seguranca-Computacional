# Descrição

O trabalho implementa a cifração, decifração, e a quebra da cifra ao achar uma chave.

# Como usar

Escreva a mensagem a ser cifrada/decifrada no arquivo ```message.txt```

Rode: 

- ```./main```

Caso precise recompilar:

- ```g++  *.cpp -o main```
